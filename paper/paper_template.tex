\begin{abstract}
Unmanned Aerial Vehicle (UAV) path planning in complex 3D environments 
remains a challenging optimization problem due to multiple constraints 
including obstacle avoidance, altitude limits, and path smoothness. 
While swarm intelligence algorithms have shown promise, newly proposed 
algorithms often struggle when directly applied to constrained 
path planning tasks. 

This paper investigates the application of the Crested Porcupine 
Optimizer (CPO), a recently proposed nature-inspired metaheuristic 
(2024), to UAV path planning in 3D terrain environments. 
We analyze CPO's limitations in this constrained setting and propose 
an Improved CPO (ICPO) that incorporates:
(1) a personal best memory mechanism to preserve search history, and
(2) an adaptive exploration-exploitation balance strategy.

Extensive experiments across five algorithms (SPSO, GWO, WOA, CPO, ICPO) 
on benchmark terrain maps demonstrate that: (i) ICPO achieves a 22.9\% 
improvement over original CPO (mean cost 6372 vs.\ 8267); (ii) GWO 
emerges as a strong baseline (mean 4932), statistically tied with SPSO 
(mean 4802, $p=0.18$ by Wilcoxon test); (iii) ablation confirms the 
adaptive exploration strategy as the dominant contribution (10.7\%), 
with pBest memory adding incremental gains (2.5\%).
\end{abstract}

\section{Related Work}

\subsection{Swarm Intelligence for Path Planning}
% PSO, GWO, WOA, ACO for path planning
Swarm intelligence (SI) algorithms have been widely applied to UAV path planning 
due to their ability to handle nonlinear, non-convex optimization problems 
\cite{karaboga2012}.

Particle Swarm Optimization (PSO) \cite{kennedy1995} and its variants are 
among the most popular. The Spherical Vector-based PSO (SPSO) \cite{spsp} 
encodes paths using spherical coordinates $(r, \psi, \phi)$, transforming 
the constrained path planning into an unconstrained optimization in 
spherical space. NMOPSO \cite{nmopso} extended this with multi-objective 
optimization incorporating kinematic constraints.

Grey Wolf Optimizer (GWO) \cite{mirjalili2014} has also been applied, 
with IGWO \cite{igwo} recently proposing cooperative predation strategies 
and lens opposition-based learning for UAV path planning. Whale Optimization 
Algorithm (WOA) \cite{mirjalili2016} has seen improvements like CICDWOA 
\cite{cicdwoa} with Cauchy-based strategies for 3D path planning.

Ant Colony Optimization (ACO) \cite{dorigo2006} has been used for multi-UAV 
cooperative inspection \cite{aco3d}. However, most existing works focus on 
well-established algorithms; newly proposed metaheuristics remain largely 
untested on constrained path planning problems.

\subsection{Crested Porcupine Optimizer}
CPO \cite{cpo} is a recently proposed (2024) nature-inspired metaheuristic 
that simulates four defense strategies of crested porcupines: visual 
deterrence, sound deterrence, physical attack, and lethal attack. CPO 
has demonstrated competitive performance on CEC benchmark functions 
against state-of-the-art algorithms including PSO, GWO, and WOA. 
However, its application to constrained engineering problems such as 
UAV path planning has not been investigated, which motivates this work.

\subsection{Research Gap and Contributions}
While SI algorithms have been extensively studied for path planning, 
most work focuses on established algorithms (PSO, GWO, WOA, ACO). 
Newly proposed algorithms like CPO remain unexplored in this domain. 
Furthermore, when directly applied, many SI algorithms struggle due to 
the highly constrained nature of 3D path planning. This paper bridges 
this gap by: (1) providing the first systematic evaluation of CPO on 
constrained UAV path planning, (2) identifying key limitations of CPO 
in this setting, and (3) proposing targeted improvements (ICPO) that 
significantly enhance performance.

\subsection{UAV Kinematic Model and Path Representation}
% Spherical vector encoding from SPSO framework
% Each path node: (r_i, \psi_i, \phi_i) → (x_i, y_i, z_i)

\subsection{Cost Function}
\begin{equation}
    J = b_1 J_{\text{length}} + b_2 J_{\text{threat}} + b_3 J_{\text{altitude}} + b_4 J_{\text{smooth}}
\end{equation}

Where:
\begin{itemize}
    \item $J_{\text{length}}$: Total path length (sum of Euclidean distances)
    \item $J_{\text{threat}}$: Collision penalty with cylindrical obstacles
    \item $J_{\text{altitude}}$: Altitude constraint penalty
    \item $J_{\text{smooth}}$: Turning and climbing angle penalty
    \item $b_1=5, b_2=1, b_3=10, b_4=1$: Weight coefficients
\end{itemize}

\section{Proposed Method: Improved CPO (ICPO)}

\subsection{Original CPO Algorithm}
% Four defense mechanisms:
% 1. Visual deterrence (exploration)
% 2. Sound deterrence (exploration)  
% 3. Physical attack (exploitation)
% 4. Lethal attack (exploitation)

\subsection{Limitations of CPO in Constrained Path Planning}
\begin{enumerate}
    \item \textbf{No personal best memory}: Unlike PSO which tracks pBest for 
          each particle, CPO only stores the previous position. This causes 
          loss of good solutions when the greedy selection rejects a newly 
          generated but worse position.
    \item \textbf{Purely random strategy selection}: CPO uses $rand() < rand()$ 
          for exploration/exploitation switching, providing no adaptive control 
          over the search behavior across iterations.
    \item \textbf{Fitness-weighting instability}: The fitness-weighted step 
          $S_t = \exp(f_i / \sum f_j)$ can produce extreme values when the 
          cost distribution is skewed by infeasible solutions.
\end{enumerate}

\subsection{ICPO Improvements}

\subsubsection{Personal Best Memory}
Each agent $i$ maintains a personal best position $\mathbf{pBest}_i$:
\begin{equation}
    \mathbf{pBest}_i = \arg\min_{\tau \leq t} J(\mathbf{X}_i^\tau)
\end{equation}

In all four defense strategies, random reference agents use their pBest 
positions instead of current positions for guided search.

\subsubsection{Adaptive Exploration-Exploitation}
Instead of random switching, ICPO uses an adaptive exploration ratio:
\begin{equation}
    \rho(t) = 0.7 \times (1 - t/T_{\max})^{0.5} + 0.3
\end{equation}

Early iterations favor exploration ($\rho \approx 1.0$), 
gradually shifting to exploitation ($\rho \rightarrow 0.3$).

\subsection{Discussion}

Table~\ref{tab:results} summarizes 20-run comparisons across all five algorithms.
Key findings:

\textbf{Top tier -- SPSO \& GWO:} SPSO (mean 4802) and GWO (4932) show no
statistically significant difference (Wilcoxon $p = 0.18$), despite GWO using
only 150 particles vs.\ SPSO's 500 and running $4.4\times$ faster.

\textbf{ICPO improvement:} Our ICPO achieves mean 6372, a $22.9\%$ improvement
over original CPO (8267), significant at $p < 0.01$ (Wilcoxon). ICPO also
significantly outperforms WOA (7501, $p < 0.01$).

\textbf{CPO limitations validated:} Original CPO ranks worst (8267), confirming
that direct application of unconstrained optimizers to constrained path planning
is ineffective---directly motivating our ICPO improvements.

\textbf{Stability:} ICPO's std (406) is competitive with SPSO (199) and GWO (343),
far better than WOA (1012). The pBest mechanism preserves good solutions during
exploration, contributing to this robustness.

\subsection{Cross-Map Validation}

We further evaluate algorithms on an urban canyon terrain with 10 dense
building-like obstacles (Table~\ref{tab:urban}). On this map, all algorithms
achieve similar performance (4666--4893), with GWO ranking first (4666) and
CPO surprisingly competitive (4674). This contrasts sharply with the mountain
terrain results, where CPO is the worst performer (8267). The disparity
confirms that CPO's limitations manifest specifically in highly constrained,
irregular terrains—precisely the scenarios where robust path planning is
most critical.

\subsection{Multi-Map Evaluation}

Table~\ref{tab:multimap} presents results across five diverse terrain scenarios.
Key observations:

\textbf{ICPO competitiveness on specific terrains:} On Map~3 (high altitude, sparse
obstacles), ICPO achieves the best mean cost (4450), outperforming both SPSO
(4466) and GWO (4480). On Maps~4--5, all three algorithms converge to identical
optimal paths.

\textbf{SPSO/GWO dominance on complex maps:} On the dense urban Map~2,
SPSO (5075) and GWO (5163) significantly outperform ICPO (5896), suggesting
that velocity-based and hierarchy-based search strategies handle cluttered
environments more effectively.

\textbf{Robustness:} ICPO consistently produces feasible paths across all maps,
validating the spherical vector encoding's generality.

\section{Conclusion}
This paper investigated the application of the Crested Porcupine Optimizer 
to 3D UAV path planning and identified key limitations when directly applying 
unconstrained optimizers to constrained problems. 
The proposed ICPO, with personal best memory and adaptive exploration, 
demonstrated significant improvement over the original CPO.

Future work includes multi-UAV coordination, dynamic obstacle environments, 
and integration with deep learning for environment perception.
